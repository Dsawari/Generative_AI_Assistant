# Generative AI Assistant & Prompt Engineering Project

## Project Description
This project explores practical applications of generative artificial intelligence, focusing on prompt engineering and API-based interaction with large language models. The work demonstrates how structured user prompts influence the quality of AI-generated responses in both text generation and image concept design tasks.

The project includes development of a simple role-based virtual assistant and experimentation with generating product design concepts using descriptive prompts.

## Technology Stack
- Python 3
- requests (HTTP API communication)
- json (data handling)
- OpenRouter API
- owl-alpha language model
- Generative image prompt evaluation (Nano Banana Pro)

## Implemented Features
- API-based connection to a generative language model
- Console-based chatbot interface
- Role-based assistant with predefined mood and character
- Dynamic behavior changes triggered by specific keywords in user input
- Dialogue prompt generation and response handling
- Experimentation with prompt engineering for improving AI output quality
- Development of structured prompts for image generation based on product descriptions
